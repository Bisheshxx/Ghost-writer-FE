# Job Tracker Feature Rules

- Keep the job tracker dashboard inside `features/job-tracker/`.
- Put UI in `features/job-tracker/component/`.
- Keep app routes thin: import the feature entry point from `features/job-tracker/index.tsx`.
- Do not add new job-tracker UI to the top-level `components/` directory.
